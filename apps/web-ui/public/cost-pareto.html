The headline is the denominator

~$0.005–$0.015 per instance. A full sweep of all 300 SWE-bench Lite problems costs roughly $1.50 in model spend for a single conformant trajectory, and ~$4.50 for the Best-of-3 + judge pipeline (measured OpenRouter spend).

The systems above it run an estimated $0.1–$2.5+ per instance; the heaviest multi-agent frontier scaffolds are reported at $15–$30/instance — $4,500–$9,000 to evaluate the same 300 problems once. That's a two-to-three-order-of-magnitude efficiency gap at comparable resolve, and it holds regardless of minor percentage swings — because the lever here is the harness (a stateful interactive loop gated by the repo's own tests), not brute-force MoE compute. Cost is the moat; resolve % is the table stakes. (Darwin costs measured; competitor costs estimated from disclosed models — see notes below.)

How to read this. The cost-Pareto frontier rewards the most resolve per dollar, not the highest absolute score. Darwin entries are measured on the full 300-instance SWE-bench Lite set (or noted pilots) with the official swebench Docker harness, fully conformant — the solver never sees gold FAIL_TO_PASS/PASS_TO_PASS tests during solving; gold is used only for final scoring. Honesty notes. Official entries' resolve % are REAL — pulled from the swe-bench/experiments repo (resolved/300). Their cost/instance is ESTIMATED from each system's disclosed model(s) + scaffold shape × public token pricing — not a figure those teams published; treat as order-of-magnitude (marked est). "undisclosed" = no public model/cost. Darwin entries are MEASURED (resolve via the official harness; cost from actual OpenRouter spend). Best-of-3 is a PILOT n=25 until the full-300 eval prints.
